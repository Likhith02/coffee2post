# Coffee2Post

Coffee2Post is a lightweight web app that turns a rough 30-second thought into a polished LinkedIn draft.

## Features

- Paste raw text or capture voice input
- Choose a vibe: casual, professional, or funny
- Choose output length: short, medium, or long
- Generate post text plus hook options
- Copy post with one click

## Run locally

```powershell
python -m http.server 8000
```

Then open:

- [http://127.0.0.1:8000/coffee2post-standalone/](http://127.0.0.1:8000/coffee2post-standalone/)
